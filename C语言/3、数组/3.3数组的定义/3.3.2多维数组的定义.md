## 格式:
```
二维数组
数据类型 数组名 [行的个数][列的个数];
int a[4][5];
定义了20个int类型的变量,分别是:
a[0][0],a[0][1],a[0][2],a[0][3],a[0][4];
a[1][0],a[1][1],a[1][2],a[1][3],a[1][4];
a[2][0],a[2][1],a[2][2],a[2][3],a[2][4];
a[3][0],a[3][1],a[3][2],a[3][3],a[3][4];
```
多维数组定义:
```
int a[3][4][5];
int a[8][3][4][5];

可以只初始化一部分
int b[][3] ={
	{1, 2, 3},
	{4, 5},
	{17, 8, 9},
	{10, 11, 12},
	{13, 14, 15}
}
```
